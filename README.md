# Expense Tracker App
A simple and efficient expense tracking app built with Flutter, featuring a minimal UI design and powered by the Isar local database.

## Features
- Add, Update, and Delete Expenses: Easily manage your daily expenses.
- Monthly Expense Overview: View and calculate total expenses per month.
- Minimalist UI: Clean and user-friendly interface for easy navigation.
- Local Database: Data is stored locally using Isar for high-performance and efficient queries.
## Tech Stack
- Flutter: Frontend UI framework
- Dart: Programming language
- Isar: Local NoSQL database
